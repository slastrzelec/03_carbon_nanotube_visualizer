# ⚛️ Carbon Nanotube Visualization

Interactive 3D visualization of different types of carbon nanotubes built with Python, Matplotlib, and Streamlit.

---

## Overview

This project allows users to explore and visualize **carbon nanotubes** of different types:

- **Armchair** – atoms arranged horizontally to the tube axis, often metallic.
- **Zigzag** – atoms form a zigzag pattern; conductivity depends on diameter.
- **Chiral** – spiral arrangement of atoms; usually semiconducting.

Users can interactively adjust the nanotube structure, rendering style, and viewing angles via a sidebar in Streamlit.

---

## Features

- **Interactive 3D visualization** of carbon nanotubes.
- **Customizable structural parameters**:
  - Radius
  - Number of atoms per ring
  - Number of rings
  - Nanotube type (armchair, zigzag, chiral)
- **Different rendering styles**:
  - Ball and Stick
  - Wireframe
  - Space Filling
  - Stick
  - Points
- **Adjustable viewing angles** (elevation and azimuth).
- **Real-time statistics** of the structure:
  - Number of atoms
  - Radius
  - Atoms per ring
